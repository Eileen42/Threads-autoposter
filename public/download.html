Windows 64bit 설치 프로그램
다운로드하고 더블클릭만 하면 끝.

⬇️ ThreadsAutoPoster-Setup-1.0.0.exe 받기 88MB · Windows 10/11 64bit

설치 흐름

  1. 위 버튼 클릭 → 다운로드 받은 .exe 파일 더블클릭
  2. 설치 마법사 → "다음" 누르며 진행 → 자동 설치
  3. 설치 완료 후 자동 실행 → 시스템 트레이(우측 하단)에 보라색 아이콘 등장
  4. Windows 시작 시 자동 실행 등록됨 (트레이 메뉴에서 끌 수 있음)
  5. 이 페이지(https://threads-autoposter.vercel.app) 새로고침 → 사용 시작
⚠️ 첫 실행 시 Windows Defender SmartScreen 경고가 뜰 수 있습니다 → "추가 정보" → "실행"
⚠️ 첫 실행 시 Playwright Chromium(약 200MB) 자동 다운로드 — 수 분 소요 (트레이 메뉴 → "상태/로그 창"으로 진행 확인 가능)
